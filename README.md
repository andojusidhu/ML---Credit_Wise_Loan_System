# ML---Credit_Wise_Loan_System
## 💳 CreditWise Loan Approval System

SecureTrust Bank provides personal and home loans across India, processing hundreds of applications daily. Currently, loan approvals rely on a manual verification process, making it time-consuming, inconsistent, and prone to bias.

This leads to:
- Rejection of eligible (low-risk) customers  
- Approval of high-risk customers, causing financial losses  

## 🚀 Objective
Build an intelligent loan approval system using Machine Learning to predict whether a loan should be Approved or Rejected based on applicant data.

## 🧠 Approach
- Data Preprocessing & Feature Engineering  
- Exploratory Data Analysis (EDA)  
- Classification Algorithms (Logistic Regression, Decision Tree, etc.)  
- Model Evaluation & Optimization  

## 📂 Dataset
Each row represents a loan applicant with:
- Personal details  
- Financial information  
- Credit history  

## 📈 Outcome
- Faster and automated loan decisions  
- Reduced risk and improved accuracy  
- Unbiased and data-driven approvals  

## 🛠️ Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib
